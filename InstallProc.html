Installation Procedures for SupaDupa999

Step 1: Get and set up the necessary files
  1. Download SupaDupa999Install.zip
    Unless otherwise directly, it will probably go to the Download folder
  2. Unzip the file into a folder of your choice
    somewhere easy to find when uploading and downloading with a file search panel
  3. Copy the Start999_encrypted - Shortcut icon in the folder SupaDupa999Install > Admin > Start
    and paste in on the desk top of your computer
    This is your Starter Page Icon
Clicking on the Starter Page Icon should initialize the Encrypted Starter Page,
The password is the one for encrypted messages between Harvey, Dad and Grandpa
Once opened, this is the Starter Page, which is the control panel for SupaDupa999

Step 1.a. Test all files downloaded
The folders you have unzipped are

    Admin
    • Start999_Encrypted-shortcut The short cut icon to the encrypted page
    • Start999_Encrypted.html The encrypted starter page, control panel
    • Start999_Plain.zip The original plain html file in an encrypted zip file (archive for future editing)
    • Note: all encryption for SupaSupa999 uses the same password
    Files999
      Archive : a folder to archive files Created in the project. Currently empty
      Templates: a folder to store html templates, imported or created. Currently 1 file
      • TSimple.html: The simplest of html templates, for lesson 1
      Temporary: a folder for temporary files, used during uploads, downloads, and edits. Currently empty

Step 1.b. Test Starter Page installation

  1. Click on the Starter Page Icon Which you have copied to the computer desk top
  2. Enter password and click Submit
  3. If you get the control panel, you have correctly installed this file

Step 2: Install the email account linked to SupaDupa999

  • Access to the email account is required for authentification at sign in
  • It should be on IPhine, email client of the computer, or on one near by
  • Email user and password are in the Encrypted Starter Page

Step 2b.a. Testing Email installation

  • Send an email to the installed address
  • If the email arrives, then you have correctly installed the email.

Step 3: Access to SupaDupa999 (Testing as well as any session)

  1. Activate email client to have easy access to email account
  2. Activate your starter Page by clicking on its icon on the desk top
  3. Open the Start999 page by entering the password and click submit button
  4. Click the link to SupaDupa999
  5. Click Sign in button top right of the page
    • Enter email name harvey@amchang.net
    • Enter github password (copy from the starter file, and paste into the required textbox
    • Github will send an authentification number to harvey@amchang.net
    • Copy the number from the email and paste it to the sign in textbox
    • You should be in
  6. Explore the site. Providing you do not edit or remove anything you cannot do harm
  7. Sign out
    • Click on the avatar icon at the top right of the main page
    • Scroll down the drop down menu to the bottom, and click Sign Out

Step 4: Tweak browser
When running github, often there are uplaoads and downloads between github and the computer

Most browsers will automatically download to the Downloads folder
This means that Harvey will then have to move the files around on the computer using File Explorer.

Doing this frequently will distract Harvey while writing his codes.

To make it easier, the browser preference should be set to Ask user to direct download folders
